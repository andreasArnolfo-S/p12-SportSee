CardProps

CardProps is an interface with properties title, value, unit, icon, bgColor and color.

Name Type Description
value number value to display on card
unit string unit of value (g, kCal, etc.)
title string title of card
icon JSX.Element icon to display on card
bgColor string background color of icon
color string color of icon
